# Smart Expense Tracker

A simple and interactive **expense management web application** built with **HTML, CSS, and JavaScript**.

Smart Expense Tracker allows users to record daily expenses, organize them by category, calculate total spending automatically, and manage their expenses through a clean and user-friendly interface.

---

## Project Overview

Managing daily expenses manually can be difficult. This project provides a simple solution where users can add, view, and delete expenses while keeping track of their total spending.

This project was developed to practice frontend web development concepts including DOM manipulation, form handling, JavaScript logic, and browser local storage.

---

## Features

 Add new expenses with details:
- Expense name
- Amount
- Category
- Date

- Display expenses dynamically  
- Calculate total expenses automatically  
- Delete unwanted expenses  
- Form validation for user inputs  
- Store expense data using browser Local Storage  
- Responsive and clean user interface
